Welcome to the IT-Repo Wiki

This site contains documentation, articles, and assets for the IT-Repo project. Use the navigation above to explore resources.

  • Getting Started: Read the guide to understand the repo structure and contribution process.
  • Articles: Visit All Articles for technical documentation and guides.
  • Images: Browse Images for diagrams and visual assets.
  • Data: Access Data for datasets and structured information.

How to Add New Content

  1. Copy docs/templates/page-template.html to docs/articles/your-article.html.
  2. Edit the placeholders ({{PAGE_TITLE}}, {{PAGE_DESCRIPTION}}, {{PAGE_CONTENT}}) with your content.
  3. Link your new page from this index or other articles.

Security & Contribution

Only trusted contributors should merge changes. Please follow the repository's guidelines and do not share sensitive information.
